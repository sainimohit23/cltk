# Usage Example

```
from sandhisplitter import SandhiSplitter
s = SandhiSplitter("""पुद्गलधर्मनैरात्म्ययोर् अप्रतिपन्नविप्रतिपन्नानाम्\nअविपरीतपुद्गलधर्मनैरात्म्यप्रतिपादनार्थं\nत्रिंशिकाविज्ञप्तिप्रकरणारम्भः  \nपुद्गलधर्मनैरात्म्यप्रतिपादनं पुनः क्लेशज्ञेयावरणप्रहाणार्थम्""")
print(s.getSandhi())
```
